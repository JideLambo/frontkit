Labels

Give your components some labels for easily distinguish them from the others.

# Label Options

Below are exemplified all available options of labels in Frontkit. Switching the styles is as easy as changing the label-* class suffix.

Default Success Info Warning Danger 
<!-- Default label -->
<span class="label label-default">Default</span>

<!-- Success label -->
<span class="label label-success">Success</span>

<!-- Info label -->
<span class="label label-info">Info</span>

<!-- Warning label -->
<span class="label label-warning">Warning</span>

<!-- Danger label -->
<span class="label label-danger">Danger</span>
