Nav

Navs are navigation components. They share a common markup and a common base .nav class, what makes it easier to switch styles.

# Nav Lists

Nav lists are vertical navigations that have light styles.

  • Header
  • Item
  • Active Item
  • Item
<ul class="nav nav-list">
    <li class="nav-header">Header</li>
    <li class="nav-item"><a href="#">Item</a></li>
    <li class="nav-item active"><a href="#">Active Item</a></li>
    <li class="nav-item"><a href="#">Item</a></li>
</ul>
