Browser and Device Support

Frontkit support a wide range of browsers, but obviously we have some limitations.
You can check this page to see the supported browsers, their version and the OS.

# Supported Browsers

We support the last 2 versions of each browser below. However, for Internet Explorer, the support is wider: from version 9 and up. This is due to the still high percentage of IE9 users compared to the other versions.

Chrome Firefox Internet Explorer Safari
Android ✔ ✔ N/A N/A
iOS ✔ N/A ✔
Linux ✔ ✔ N/A
Mac OS X ✔ ✔ ✔
Windows ✔ ✔ ✔ - IE9+ ✘

Other browsers may work as well, however, we won't officially cover them.
