Responsive Utilities

Frontkit provides you a lot of powerful classes to deal with eventual differences between screen sizes.

Visibility

You can show or hide a element depending on the screen size in a ton of ways. Check both tables below to discover how you can do this.

Visible by size

The below table shows classes that are used to make elements visible depending on the screen size.

≥ 768px 		Medium
≥ 980px 		Large

Orientation

A simple way to detect the orientation of your device. On mobile devices, the visibility of elements will automatically change whenever you rotate the device. On desktops, the orientation is almost always landscape.
