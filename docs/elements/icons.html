Icons

Frontkit includes a subset of icons of the WebHostingHub Glyphs iconic font, which makes easy for you to use flat, vectorized, retina ready icons.

# Usage

To use a Frontkit icon, you must use the .icon class in conjunction with the icon class, available below: 

<span class="icon icon-home"></span>
