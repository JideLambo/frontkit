Images

Rounded

An image can be rounded when you add the img-rounded class. By default, any image will get a 4% border-radius, no matter how big it is. 

<img src="..." class="img-rounded">

Circular

Images will be circular if you add the img-circular class.

<img src="..." class="img-circular">

Thumbnail

A image can be bordered by simply adding the img-bordered class.

<img src="..." class="img-bordered">
